# Cross-Border Logistics

## Shipping Methods
- FBA: Best for top-selling ASINs (Prime badge)
- FBM: Best for testing, oversized items
- Hybrid: FBA for best-sellers, FBM for slow movers

## Customs Tips
- Correct HS Code classification is critical
- Include: commercial invoice, packing list, B/L
